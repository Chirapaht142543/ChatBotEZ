ปัญญาประดิษฐ์ตอบแทน (Gemini AI Integration)

กำลังตรวจเช็คคีย์ของ Gemini AI...

หากตั้งค่าคีย์ API ในระบบหลังบ้าน (Environment) บอทจะสามารถประมวลผลข้อความเพื่อสนทนากับลูกค้าได้อย่างต่อเนื่อง

เปิดใช้งาน AI ตอบกลับอัตโนมัติ (Gemini / Claude) เมื่อเปิดใช้งาน บอทจะใช้ AI ประมวลผลก่อนเป็นอันดับแรก หากไม่พบคีย์หรือไม่มีคำตอบจะข้ามไปใช้กฎคีย์เวิร์ด
กำหนดบุคลิกภาพ วิธีการพูดจา และข้อมูลข้อจำกัดของบอทเพื่อให้บริการอย่างสุภาพ
เมื่อลูกค้าพิมพ์ข้อความตรงกับคำในนี้ บอทจะหยุดตอบและระบบจะโยนเคสให้แอดมินคนจริงเข้าควบคุมทันที

คู่มือแนะแนวการเปิดใช้งาน Gemini API

  • 1
    สร้างคีย์ API ฟรีได้ง่ายๆ ที่ Google AI Studio ด้วยบัญชี Google ของคุณ
  • 2
    คัดลอกคีย์ความลับ (API Key) ที่สร้างขึ้นมา
  • 3
    กำหนดคีย์นี้ให้เซิร์ฟเวอร์โดยตั้งค่าตัวแปรสภาพแวดล้อม GEMINI_API_KEY
  • 4
    รัน HappyBot ด้วยสคริปต์ (เช่น ใน PowerShell):
    $env:GEMINI_API_KEY="AIzaSy..." ; node server.js

เพิ่มกฎการตอบกลับ

หากลูกค้าส่งข้อความมีคำตรงกับคีย์เวิร์ดในนี้ บอทจะเลือกกฎนี้ไปตอบกลับ (ไม่สนพิมพ์เล็ก-ใหญ่)
ใช้ภาษาที่เป็นกันเอง สามารถใช้อีโมจิเพื่อเพิ่มความเป็นมิตรได้ 😊
