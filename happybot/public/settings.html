ตั้งค่าระบบ (System Settings)

ควบคุมพฤติกรรมของ AI เปลี่ยนแปลงข้อมูลโปรไฟล์ เชื่อมต่อ API ภายนอก และจัดการสิทธิ์ทีมแอดมิน

ข้อมูลร้านค้าและแบรนด์

ตั้งค่ารายละเอียดร้านค้าเบื้องต้นเพื่อใช้เป็นบริบทในการตอบคำถามของระบบและใบคำสั่งซื้อ

ข้อความเริ่มต้นที่จะแสดงในหน้าต่างแชทฝั่งลูกค้าโดยอัตโนมัติ

ระบบบอท & AI อัจฉริยะ (Gemini AI)

เปิดการทำงานของการประมวลผลคำตอบอัตโนมัติด้วย AI หรือคีย์เวิร์ด และปรับแต่งบุคลิกภาพบอท

🤖

กำลังตรวจสอบ API Key...

ระบบจะใช้ Gemini AI หากตรวจพบไฟล์สภาพแวดล้อมเพื่อสร้างความสุภาพและฉลาดตอบคำถาม

เปิดใช้ AI ตอบกลับอัตโนมัติ ให้ Gemini AI เป็นผู้ประเมินและสนทนากับลูกค้าเป็นลำดับแรก หากบอทไม่แน่ใจจึงโยนให้แอดมินคนจริงตอบ
คีย์นี้จะบันทึกแบบปลอดภัย หากรันผ่าน Environment Key (GEMINI_API_KEY) จะถือว่าเป็นค่าเริ่มต้น
ตรงประเด็น (0.0) สร้างสรรค์ (1.0)

เชื่อมต่อช่องทาง LINE Official Account

นำข้อมูลรับรองจาก LINE Developers Console มาผูกเพื่อเริ่มคุยกับลูกค้าผ่านแอป LINE จริง

นำลิงก์ Webhook URL นี้ไปตั้งค่าในหน้า LINE Developers เพื่อให้ระบบรับข้อความแชทเข้ามาได้

เชื่อมต่อเว็บไซต์ (Website Integration)

ฝังกล่องแชทบอทลงบนเว็บไซต์ของคุณ เพื่อให้ลูกค้าสามารถสอบถามสินค้าหรือบริการได้ทันทีผ่านหน้าเว็บจริง

🌐

วิธีติดตั้ง Chat Widget ลงเว็บไซต์

คัดลอกโค้ดด้านล่างแล้ววางไว้ก่อนแท็กปิด </body> ของทุกหน้าเว็บที่ต้องการให้แสดงกล่องแชท ระบบจะแสดงปุ่มแชทมุมขวาล่างโดยอัตโนมัติ

<!-- HappyBot Chat Widget -->
<script src="http://localhost:3000/widget.js" data-color="#d4a017" data-position="bottom-right" data-greeting="สวัสดีค่ะ มีอะไรให้ช่วยไหมคะ? 💬"></script>
วางโค้ดนี้ไว้ก่อนแท็กปิด </body> ในไฟล์ HTML หรือ Theme ของเว็บไซต์
ระบุโดเมนเว็บไซต์ของคุณเพื่อให้ระบบรู้จักและอนุญาตการเชื่อมต่อ
ระบุโดเมนที่อนุญาตให้ฝัง Widget หลายโดเมนคั่นด้วยจุลภาค — ป้องกันไม่ให้เว็บอื่นนำ Widget ไปใช้โดยไม่ได้รับอนุญาต
ตั้ง 0 เพื่อไม่เปิดอัตโนมัติ หรือใส่จำนวนวินาทีที่ต้องการ
ข้อความฟองที่จะแสดงเหนือปุ่มแชทเพื่อดึงดูดลูกค้า
สวัสดีค่ะ มีอะไรให้ช่วยไหมคะ? 💬

การตั้งค่าการแจ้งเตือน

เปิดรับการเตือนในรูปแบบต่างๆ เพื่อไม่ให้พลาดทุกแชทสำคัญจากลูกค้า

เปิดเสียงแจ้งเตือนข้อความแชทใหม่ เล่นเสียงคลิกสั้นๆ เมื่อแอดมินเปิดหน้าเว็บอยู่แล้วมีข้อความใหม่จากลูกค้าเด้งเข้ามา
การแจ้งเตือนบนหน้าจอคอมพิวเตอร์ (Desktop Notifications) ส่ง Notification แบนเนอร์ขนาดเล็กผ่านเบราว์เซอร์เมื่อคุณพับหน้าจออยู่
แจ้งเตือนแชทค้างผ่านอีเมล ส่งสรุปอีเมลเตือนความจำกรณีมีแชทลูกค้าตกค้างนานเกิน 15 นาทีโดยไม่มีการตอบกลับ
