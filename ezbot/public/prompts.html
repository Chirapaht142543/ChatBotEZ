ตั้งค่า Prompt AI แยกช่องทาง

ปรับแต่งบุคลิกภาพ คำสั่งแอดมินบอท และข้อมูลโปรโมชั่นแยกเฉพาะแต่ละช่องทางของแชท

เปิดใช้งาน AI ตอบกลับอัตโนมัติ (Global AI Switch)

สวิตช์หลักสำหรับเปิด/ปิดการทำงานของบอท AI ทุกช่องทางพร้อมกันในระบบ

เปิดใช้งานบอท AI สำหรับช่องทางนี้

เปิดใช้งานเฉพาะของช่องทางนี้ (ทำงานเมื่อสวิตช์บอทภาพรวมด้านบนเปิดอยู่)

กำหนดบุคลิกภาพ วิธีพูดคุย และคำลงท้ายของ AI สำหรับช่องทางนี้โดยเฉพาะ
ระบุรายละเอียด ราคา และสเปคสินค้าที่ต้องใช้ในการตอบลูกค้าช่องทางนี้
กำหนดโค้ดส่วนลด และกิจกรรมส่งเสริมการขายเฉพาะกลุ่มลูกค้าของช่องทางนี้
บันทึกข้อมูลเรียบร้อยแล้ว!

กำลังตรวจเช็คคีย์ AI...

กรุณาตั้งคีย์ API หลังบ้านเพื่อให้ฟีเจอร์ AI ตอบกลับข้อความทำงานได้สมบูรณ์

คู่มือแนวทางการเปิดใช้งาน Gemini API

  • 1
    สร้างคีย์ API ฟรีได้ง่ายๆ ที่ Google AI Studio ด้วยบัญชี Google ของคุณ
  • 2
    คัดลอกคีย์ความลับ (API Key) ที่สร้างขึ้นมา
  • 3
    กำหนดคีย์นี้ให้เซิร์ฟเวอร์โดยตั้งค่าตัวแปรสภาพแวดล้อม GEMINI_API_KEY
  • 4
    รัน EZ BOT ด้วยสคริปต์ (เช่น ใน PowerShell):
    $env:GEMINI_API_KEY="AIzaSy..." ; node server.js
